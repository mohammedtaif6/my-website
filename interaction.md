# Dashboard Interaction Design

## 5 Main Cards:

1. **عدد المشتركين (Total Subscribers)**
   - Click to view detailed subscriber list
   - Hover shows growth percentage
   - Interactive chart showing subscriber trends

2. **المنتهي اشتراكهم (Expired Subscriptions)**
   - Click to view expired subscribers list
   - Hover shows expiration rate
   - Quick action buttons for renewal reminders

3. **على وشك الانتهاء (Expiring Soon)**
   - Click to view subscribers expiring within 30 days
   - Hover shows countdown timer
   - Send notification buttons

4. **الديون (Debts/Outstanding Payments)**
   - Click to view detailed debt list
   - Hover shows total amount
   - Payment tracking and status updates

5. **التقارير (Reports)**
   - Click to generate various reports
   - Hover shows last report date
   - Download options and filters

## Interactive Features:
- Real-time data updates
- Modal windows for detailed views
- Filter and search functionality
- Export capabilities
- Notification system
- Quick action buttons
- Responsive design for all devices
