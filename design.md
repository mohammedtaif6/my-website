# Dashboard Design Style

## Design Philosophy
- Modern SaaS dashboard aesthetic
- Clean, professional interface
- Arabic/English bilingual support
- High-contrast accessibility
- Smooth animations and transitions

## Color Palette
- Primary: Deep blue (#1e40af) and purple (#7c3aed)
- Secondary: Teal (#0d9488) and orange (#ea580c)
- Neutral: Gray scale (#374151, #6b7280, #f9fafb)
- Success: Green (#059669)
- Warning: Orange (#ea580c)
- Error: Red (#dc2626)

## Typography
- Primary font: Inter (Arabic/English)
- Headers: Bold weights (600-700)
- Body: Regular weights (400-500)
- Numbers: Tabular figures for alignment

## Visual Effects
- Subtle gradients on cards
- Hover animations with scale and shadow
- Loading states with skeleton screens
- Smooth transitions (300ms ease)
- Floating action buttons
- Progress bars and indicators

## Layout
- Grid-based responsive design
- Card-based information architecture
- Consistent spacing (8px base unit)
- Mobile-first approach
- RTL support for Arabic content
